# Blockland Survival v0.8

Protótipo web 3D inspirado em survival blocado, pronto para GitHub Pages.

## Como publicar
Envie estes arquivos para a raiz do repositório e ative GitHub Pages.

## Controles
- WASD/Setas: andar ou dirigir
- Mouse: olhar
- Espaço: pular
- Shift: correr
- E: coletar/interagir/entrar no carro
- X: sair do carro
- F ou clique esquerdo: usar item/equipamento
- Botão direito com detonador: armar C4 mirando nela
- 1-9 e 0: hotbar
- B: usar Kit Base
- J: chat
- M: mapa
- ,: kits
- TAB: inventário/vault

## Recursos v0.7
Bases, safe zone, NPCs, XP, airdrops, C4, carros, PVP simulado, zumbis perto das casas e inventário em grade.

Sons e modelos são originais/procedurais; nenhum asset oficial é incluído.


## Balance v0.8

- Fome agora desce de 0.55/s para 0.07/s.
- Sede agora desce de 0.75/s para 0.09/s.
- Tempo aproximado até zerar: fome ~24 minutos, sede ~18 minutos.
- Objetivo: permitir PVP, raid e exploração sem precisar parar toda hora para comer/beber.
